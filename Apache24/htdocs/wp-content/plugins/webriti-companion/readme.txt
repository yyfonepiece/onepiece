Enhance Webriti WordPress Themes functionality.

== Description ==

This plugin is the companion plugin for Quality. It creates the service section and portoflio section on home page in the Quality theme.
Webriti Companion is a plugin build to enhance the functionality of WordPress Theme made by Webriti.
This plugin create repeater controls in the customizer settins allowing you to create a live site without moving out to customizer screen. Right now plugin have a support for Quality WordPress Theme. In future this plugin will support other themes by webriti.
